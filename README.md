# Dillinger
## _The Last Markdown Editor, Ever_

## init load data
```sh
$ python manage_local.py loaddata ./blog/fixtures/blog.json
```

## run test
```sh
$ pytest
```


## 실행
```sh
$ python manage_local.py runserver
```

[http://127.0.0.1:8000/](http://127.0.0.1:8000/)
